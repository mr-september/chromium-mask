Why Chromium Mask?

Even though browsers like Opera, Brave, Edge, Vivaldi, and Arc are built on Chromium, many websites still block them or provide degraded functionality. This extension solves that problem.

📦 Manual Installation

Works on any Chromium browser!

  1. Download and unzip the latest release from the Releases page
  2. Open your browser's extension management page:
    • Opera: opera://extensions
    • Brave: brave://extensions
    • Edge: edge://extensions
    • Vivaldi: vivaldi://extensions
    • Chrome: chrome://extensions
    • Arc: arc://extensions
  3. Enable "Developer mode" (toggle in the top-right corner)
  4. Click "Load unpacked" and select the unzipped extension folder
